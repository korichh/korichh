### 🛠 What I'm building on my own time

**[Treckoner](#)** — a crypto-native trading journal that tells you what you actually own and what it's worth, right now. Real-time P&L is derived entirely from your trade log (never stored, never drifts), fed by a live Binance price pipeline fanned out over Redis pub/sub.

**[Amindy](#)** — a white-label AI support chatbot platform for e-commerce. The centerpiece: admins register any REST endpoint through a form, and the agent synthesizes a real, typed LLM tool from it at runtime — no redeploy, no custom code per client.
